Basic usage

Search

Enter a keyword on search bar and select
  • Header Just search by header
  • Main Just search by main
  • Header+Main Search by header and main

New paragraph

Click button liked +, and fill header and main Valid Markdown


Drag

Click a button named Sort and drag some paragraphs by using mouse, click Complete to complete
